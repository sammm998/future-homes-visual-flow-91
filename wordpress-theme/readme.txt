A modern real estate theme for international property investment in Turkey, Dubai, Cyprus & France.

== Description ==

Future Homes Turkey is a comprehensive real estate WordPress theme designed for international property investment companies. It features:

* Property listings with detailed information
* Multi-currency support (EUR, USD, GBP, TRY)
* Property search and filtering
* AI-powered property search
* Property finder wizard
* Team member profiles
* Client testimonials
* Multilingual support
* Responsive design
* SEO optimized

== Installation ==

1. Upload the theme files to the `/wp-content/themes/future-homes-turkey` directory
2. Activate the theme through the 'Appearance > Themes' screen in WordPress
3. Configure the theme through Appearance > Customize
4. Create your property listings using the Properties post type

== Custom Post Types ==

The theme includes the following custom post types:
* Properties - Real estate listings
* Team Members - Staff profiles
* Testimonials - Client reviews

== Custom Fields ==

Property posts include the following custom fields:
* Reference Number
* Price
* Bedrooms
* Bathrooms
* Area (sqm)
* Latitude/Longitude for maps

== Menus ==

The theme supports the following menu locations:
* Primary - Main navigation
* Footer - Footer links
* Mobile - Mobile navigation
